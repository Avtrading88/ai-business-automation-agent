# Client Demo Agenda

## Goal

Show how the agent reduces manual spreadsheet work and prevents risky CRM/accounting updates.

## Agenda

| Time | Topic |
|---:|---|
| 1 min | Business problem and workflow overview |
| 2 min | Upload spreadsheet and clean data |
| 1 min | Review rejected rows and duplicates |
| 1 min | Download CRM/QuickBooks-ready outputs |
| 1 min | Approval, roles, and audit log |
| 1 min | Scheduling, notifications, backups, rollback |
| 1 min | Next integration options |

## Discovery questions for client calls

- Which CRM do you use today?
- Which accounting system do you use today?
- What spreadsheet columns are most important?
- What fields must be validated before import?
- Who should approve changes before sync?
- Do you need daily, weekly, or manual processing?
- What reports do managers need after processing?
