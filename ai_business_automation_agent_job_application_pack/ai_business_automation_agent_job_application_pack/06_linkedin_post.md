# LinkedIn Post Draft

I recently built an AI Business Automation Agent as a portfolio project.

The goal was to solve a very common business problem: messy spreadsheet data before it enters CRM or accounting systems.

The app can:
- Read CSV and Excel files
- Clean and validate business data
- Detect missing values and duplicates
- Generate clean CRM/QuickBooks-ready outputs
- Create reports and audit logs
- Add human approval before external sync
- Support role-based access, scheduled jobs, backups, and file versioning

Tech stack:
Python, Pandas, Streamlit, SQLite, Pytest, Ruff, Black, Docker, and GitHub Actions.

This project helped me practice data cleaning, automation workflows, dashboard development, testing, CI/CD, and business-focused software design.
