# GitHub README Intro

# AI Business Automation Agent

A Python-based automation dashboard for cleaning, validating, auditing, and preparing business spreadsheet data for CRM and accounting workflows.

## Business Problem

Many businesses still manage contacts, customer records, invoices, and operational data inside spreadsheets. Before this data is uploaded to CRM or accounting platforms, it often contains missing fields, invalid emails, duplicates, inconsistent formatting, and manual-entry errors.

## Solution

This project provides a safe automation workflow that reads spreadsheet data, cleans it, validates it, detects duplicates, creates clean output files, generates audit reports, and prepares CRM/QuickBooks-ready data. It also includes human approval, dry-run mode, audit logs, role-based access, scheduled jobs, and rollback features to reduce risk before syncing data with external systems.

## Tech Stack

Python, Pandas, Streamlit, SQLite, Pytest, Ruff, Black, Docker, GitHub Actions, YAML configuration, and API integration planning.
